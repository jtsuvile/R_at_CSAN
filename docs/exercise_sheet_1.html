3.1 Making vectors using functions You have already encountered two ways to make a vector: c and :. There are also other efficient commands that make vectors, including seq and rep.
You can use seq to create a sequence in two ways. Try typing

vector1 <- seq(1,20,by=0.5) vector2 <- seq(1,20,length=5) Question 10 What is the difference between the two vectors and the use of named arguments (by, length) in defining the sequence?
With rep, you can repeat values to make, for instance, a vector. Again, there are multiple ways to do this: rep(1:5, times=2) rep(1:5, times=c(3,1,1,5,2)) rep(1:5, each=3)
